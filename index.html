Basic c++

Let's print in c++!

First make sure to include iostream when printing otherwise you will not get to print stuff! Like this:


    #include <iostream>

After that start a main function this powers the whole c++ program!


    #include <iostream>
    int main() {
    }

After that you need to write something in the output to make it print!


    #include <iostream>
    int main() {
        std::cout << "Hello";
    }

I will explain this carefully so you understand.

std is standard like the standard way to use libraries.

:: is like the scope to use the libraries.

Cout is the output.

<< is like saying write or input something here!

"" is a string aka a string of letters

And finally ; is for ending a line.

Oh yeah i forgot add return 0; to say like "Hey this function passed! It was a sucsess". So now type:


    #include <iostream>
    int main() {
        std::cout << "Hello";
        return 0;
    }
